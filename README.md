### Key Features <a name="key-features"></a>

The main features of our application can be seen by our console app. Where you can:
- List all books
- List all music albums
- List of games
- List all genres (e.g 'Comedy', 'Thriller')
- List all labels (e.g. 'Gift', 'New')
- List all authors (e.g. 'Stephen King')
- Add a book
- Add a music album
- Add a game

## ğŸ’» Getting Started <a name="getting-started"></a>

### Prerequisites

You need to have the following installed:

- Ruby
- RSpec
- IRB

### Setup

Clone this repository to your desired folder:

```sh
  git clone https://github.com/Arch-Noize/ruby-capstone.git
  cd ruby-capstone
```

### Install

This repository and database don't need to be installed.

### Usage

- To run this app, use the IRB console and use
```bash
ruby main.rb
```

### Run tests

- To run tests, open a terminal or `cd` into the project's folder and run
```bash
rspec
```

### Deployment

It is not deployed in any other website or service.
